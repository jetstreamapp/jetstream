## What's new

### In-app feedback

You can now send feedback from inside Jetstream. Open the feedback widget, add a message, optionally attach files, and it reaches the Jetstream team without needing a GitHub account. The widget is available on web, desktop, and the browser extension.

### Reliable extension attachments

File attachments now upload reliably in the browser extension.

### Quality-of-life

- The IP-address banner was removed.
- Documentation grammar fixes across various pages.
