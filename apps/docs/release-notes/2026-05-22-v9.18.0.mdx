## What's new

### Refresh a record without closing the modal

When you're viewing or editing a record, you can now refresh it in place to pull the latest values. This
is especially useful when automation runs in the background and changes the record after you've opened
it — no need to close and reopen to see the current state.

### Deep-link from the browser extension into Query

When you open Jetstream from the web extension, it can now carry context with you: the query page can
auto-populate the object you came from and, where it makes sense, add a filter for that record's id. That
shaves several steps off the common "I'm looking at this record in Salesforce, now let me query around
it" workflow.

### Expanded audit logging for org actions

Actions on your Salesforce org connections — adding an org, deleting one, and refreshing its token — now
write audit log entries. That gives team owners a clearer trail of who connected or changed what.

### Other fixes

- The desktop app handles logout and concurrent-login detection more carefully, so a logout can no longer
  accidentally restore a stale authentication state.
