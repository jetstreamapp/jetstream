## What's new

### Feedback attachments

The in-app feedback widget shipped in 8.12.0 had issues when sending more than one attachment. This release:

- Caps attachments at 5 per message, with a warning toast if you try to add more.
- Buffers attachments server-side so the email provider receives every file, instead of only the first stream.
