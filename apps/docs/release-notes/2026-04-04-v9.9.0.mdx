## What's new

### Long-running requests no longer time out

Some Salesforce operations, like large bulk queries and heavy metadata reads, can take a while to finish and previously failed with a timeout error partway through. These long-running requests now stay alive until the operation completes and returns its result, so they finish instead of timing out.
