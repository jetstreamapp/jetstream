## What's new

### SOQL format options

You can now configure how Jetstream formats the SOQL it generates. Pick the casing and layout options you prefer, and the query builder will apply them consistently whenever it writes SOQL on your behalf.

### Desktop editor fix

Fixed a build issue in the desktop app where Monaco editor assets were not packaged correctly, which affected the code editor experience.
