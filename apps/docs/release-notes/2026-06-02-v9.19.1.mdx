## What's new

This is a security-focused release. It hardens the history data-sync routes and IP detection against
prototype-pollution attacks, and upgrades the underlying build tooling (Vite and Vitest) to address a
published vulnerability. There are no changes to how you use Jetstream — these are protective updates.
