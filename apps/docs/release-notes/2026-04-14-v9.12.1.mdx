## What's new

### Retry failed rows

The load-failures modal now supports row selection with a Retry Selected action. Pick the rows you want to re-run and Jetstream kicks off a new load using the existing mapping and options, without forcing you to export the failures and start over.

### CSP form-action directive

Adds `form-action` directives to the Content Security Policy so form-POST redirects to third-party origins (for example during OAuth) are not blocked.
