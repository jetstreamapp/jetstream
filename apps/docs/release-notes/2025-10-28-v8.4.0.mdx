## What's new

### View Record checkbox fix

Checkbox fields in the read-only View Record view were rendering as blank. They now show the correct true/false value. Resolves #1414.

### Desktop auto-update overhaul

The desktop app's old update flow forced you through a dialog, sometimes multiple times. Update state now lives in a popover in the navbar so you can see progress, trigger installs, and dismiss at your own pace. Shipped in desktop 3.3.0.

### Zip export filename options

When exporting binary attachments, you can pick the filename format, which avoids collisions when records share an attachment name. Resolves #1379.

### Navbar cleanup without an org

Record search and user search icons are hidden until an org is selected, so you no longer see features that cannot work. Resolves #1395.

### Desktop team login

The desktop entitlement check now prefers team entitlements when they exist and refuses login if the user is not active in the team.
