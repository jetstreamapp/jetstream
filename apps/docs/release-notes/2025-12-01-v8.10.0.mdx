## What's new

### Invalid session detection

Jetstream now recognizes additional Salesforce error messages that indicate an invalid or expired session, and marks the affected org with a connection error. This means you get the standard "reconnect org" prompt instead of mysterious request failures.

### SOQL parser 6.3.0

`soql-parser-js` was upgraded to 6.3.0, which brings broader support for recent SOQL grammar additions.

### Base64 regression coverage

The shared base64-to-ArrayBuffer helper was consolidated and now has dedicated tests so the 8.9.x download regression cannot recur silently.
