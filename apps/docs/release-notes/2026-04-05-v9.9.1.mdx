## What's new

### Cloudflare Insights allowed in CSP

Cloudflare Insights was being blocked by our Content Security Policy. The `script-src` directive now includes the Insights host so analytics load cleanly.

### SSO blog link

The landing page link to the SSO blog post was pointing at a stale route. It now resolves to the current Jetstream SSO post.
