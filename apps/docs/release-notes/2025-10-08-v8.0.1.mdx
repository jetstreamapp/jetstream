## What's new

### Fix profile parsing crash

After 8.0.0 shipped, some users hit an error on load because their profile payload did not include an `entitlements` or `preferences` object. The user-profile schema now accepts a null value for both fields and falls back to safe defaults, so the app initializes cleanly for every account.
