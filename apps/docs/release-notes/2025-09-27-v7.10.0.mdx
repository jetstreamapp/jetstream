## What's new

### Desktop history sync fix

Fixed a bug where history sync could fail to connect on the desktop app when it was enabled.

### Binary file uploads on desktop

Binary attachment uploads on the desktop app were failing. They now upload correctly.

### Outbound IP address ranges

The getting started docs now describe the upcoming API outbound IP address ranges, and the app home page shows a notice about the change.
