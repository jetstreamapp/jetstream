## What's new

### Full-name fields regenerate only when you want them to

Auto-generating a full-name field is convenient, but constantly overwriting your edits is not. Jetstream
now only auto-generates the value when the field is empty, so it won't clobber a name you've already set.
When you do want to refresh it from the underlying parts, there's a quick manual regenerate option.

### Outbound IP addresses consolidated

Jetstream's outbound traffic now comes from three specific IP addresses rather than a wide range. These
addresses fall within the range Jetstream previously published, so no action is required and connectivity
is unaffected — but if your org allowlists Jetstream by IP, you can now narrow that list from a range of
510 possible addresses down to three. See the security documentation for the current addresses.
