## What's new

### Geo IP service routing fix

The geo IP API service had a router definition that was incompatible with Express 5 after a recent upgrade. Requests now route correctly again.

### Docs and build cleanups

- The Windows desktop download link in the connected app docs has been corrected.
- A desktop build script no longer tries to uninstall dependencies that are not present, which was breaking the build.
