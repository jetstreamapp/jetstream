## What's new

### View All Records column width

The new View All Fields column introduced in 7.9.0 was causing the View All Records column in the Manage Permissions grid to render at the wrong width. The column sizing was corrected.

### Correct x64 desktop download link

The download link for the x64 desktop installer was pointing at the wrong asset. The download resolver was updated so the correct Windows x64 build is served.
