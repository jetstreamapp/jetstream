## What's new

### Profile and Permission Set info popover

Profile and Permission Set rows in Manage Permissions, Create Fields selection, Create New Object Permissions, Formula Evaluator permissions, and the User search popover now have a small info trigger. Clicking it opens a popover that shows the record label, a searchable list of assigned active users (for profiles) or active assignees (for permission sets), and a direct link to the record in Salesforce Setup.
