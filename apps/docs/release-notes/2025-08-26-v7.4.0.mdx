## What's new

### Per-org encryption for stored credentials

We moved to a stronger encryption strategy for your stored Salesforce org credentials. Each stored org now uses its own encryption key instead of a single shared key, which limits the impact if any one key were ever exposed.

No action is required. Existing orgs continue to work.
