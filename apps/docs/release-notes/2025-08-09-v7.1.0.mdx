## What's new

### Toggle unchanged lines when comparing metadata

The view-and-compare metadata modal now has a control to hide unchanged lines, so you can focus on the parts of a file that actually differ. Filename sorting in that view was also fixed so files list in a stable order.

### Folder metadata types

Document, Email, Report, and Dashboard folders are now included when you list metadata and retrieve correctly. This resolves cases where folder-based metadata was missing from the deploy or compare workflows.

### Datatable tab navigation

Fixed an edge case where Tab navigation inside the datatable could treat the action and Id columns as normal cells in production builds. Tab navigation now behaves consistently everywhere the datatable is used.

### Other fixes

- Fixed a scroll-lock glitch on the landing page, and popovers and modals now render in the right place.

### Web extension 2.0.1

The browser extension picked up the scroll-lock and popover fixes.
