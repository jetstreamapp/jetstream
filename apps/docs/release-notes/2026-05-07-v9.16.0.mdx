## What's new

### Date shortcuts when picking a date

The date picker now offers quick shortcuts for the ranges you reach for most, so filtering a query or
setting a value no longer means clicking through a calendar to get to "yesterday" or "last week."

### Deploy and compare metadata folders are more dependable

Folder-based metadata is now resolved correctly when you deploy or compare metadata. Email templates
stored inside a folder are no longer missed, and duplicate folder entries that previously caused tables
to render incorrectly (depending on which metadata types you fetched) have been sorted out.

### Behind-the-scenes security hardening

This release includes a broad round of security work that keeps your team's data safer without changing
how you use Jetstream:

- Added CSRF protection to team and billing pages.
- Sensitive routes now always read your role fresh from the database rather than trusting session data,
  and billing access is locked down so it can't reach admin-only routes.
- Tightened the SAML identity-provider-initiated login flow and added caching for it.
- Jetstream no longer sends its own cookies to Salesforce platform-event endpoints.
- Billing role changes now reflect in Stripe reliably.

### Other fixes

- Picklist values display correctly when mapping fields during a load.
- Number formatting is now handled with the platform's built-in `Intl.NumberFormat` for consistency.
- Permission filtering handles a wider range of objects.
