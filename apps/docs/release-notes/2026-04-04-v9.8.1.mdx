## What's new

### Resend invitation fix

An empty body was being sent to the `resendInvitation` API, which was rejected as invalid input. The request now includes an empty data object so invitations resend correctly.

### Bot route status code

The bot-block route now responds with HTTP 444 (connection closed without response) instead of 403. This cuts down on retry storms from abusive clients.
