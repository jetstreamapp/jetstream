## What's new

### Keyboard navigation that gets out of your way

Getting around Jetstream with the keyboard is much smoother in this release. Popovers now properly take
focus when they open (previously they often didn't, which made tabbing through them frustrating), and
focus returns where you'd expect when they close. The org-info popover and the app tab bar also got
keyboard-navigation and focus fixes.

### Smoother drag-and-drop

Drag-and-drop for organizing org groups and building query filters feels more reliable and responsive.
Moving orgs between groups and rearranging filter rows is smoother as a result.

### The browser extension stops churning tokens

The web extension's access token was expiring far more often than it should have. That's been fixed, so
the extension refreshes its token much less frequently and stays connected more steadily.

### Other fixes

- More robust copy-to-clipboard handling.
