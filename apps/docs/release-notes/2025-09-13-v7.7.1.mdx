## What's new

### Login with Salesforce URL fix

Some users reported invalid URLs when clicking "Login with Salesforce" links in Jetstream. The links now build URLs using proper URL parsing instead of string concatenation, and double slashes in the path are no longer possible.

### Attachment download host handling

Related URL handling in the attachment download flow was also hardened so the generated URLs are always well-formed and target the expected host.
