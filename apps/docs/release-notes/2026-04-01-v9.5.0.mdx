## What's new

### Salesforce formula evaluation, rebuilt

Formula evaluation has been rebuilt to cover far more of the real-world Salesforce formula language. It fixes a number of long-standing bugs where valid formulas returned wrong values or failed outright.

### Desktop persistence fixes

On some Windows VDI and virtual-machine setups, users were getting logged out or seeing their org list wiped. Both issues are now fixed, so your login and saved orgs stay put across VM snapshots and profile roaming.
