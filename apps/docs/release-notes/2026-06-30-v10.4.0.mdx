## What's new

### Preview bulk updates before saving

When you make a bulk update to records, you can now preview the proposed changes first. The preview lists each impacted record alongside its original and new values, so you can confirm exactly what will change before anything is written back to Salesforce.

### View field metadata from query results

Right-click a field's column header in query results and choose to view its metadata. A modal shows the field's details without leaving your results, which is handy when you're deciding how to filter, sort, or update a column.

### Other fixes

- The column resize handle no longer overlaps frozen column headers.
- The select-all checkbox is now centered in its header cell.
