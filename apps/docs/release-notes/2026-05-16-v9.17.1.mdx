## What's new

### Token rotation improvements reach desktop and the extension

Building on the previous release's fix for daily logouts, this update improves how the desktop app and
browser extension handle access-token rotation. Token management in the service worker and extension
settings was reworked, with clearer logging to make any future issues easier to diagnose. The result is
fewer unexpected sign-outs when you're working in the desktop app or extension.
