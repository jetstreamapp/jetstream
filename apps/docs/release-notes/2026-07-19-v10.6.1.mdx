## What's new

### Windows auto-update fix

Windows desktop users whose automatic updates stopped working after a code-signing certificate change will be back on the normal update track with this release. If your desktop app hasn't been updating, you may need to reinstall it once from the website to pick up future updates automatically.

### Security hardening on desktop and browser extension

The desktop app now opens external links and downloads more safely and uses the correct filenames for downloaded files, and the browser extension restricts message handling to trusted sources. These are behind-the-scenes safety improvements with no change to how you use the app.
