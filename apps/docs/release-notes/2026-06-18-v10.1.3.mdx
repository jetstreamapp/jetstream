## What's new

### Documentation for required Salesforce permissions

This release adds detailed documentation about the Salesforce permissions Jetstream needs, along with a
reference to how Jetstream accesses Salesforce data and APIs. It's aimed at administrators who want to
grant appropriate, least-privilege access while staying aligned with their security practices.
