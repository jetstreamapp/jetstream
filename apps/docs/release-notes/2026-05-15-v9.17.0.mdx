## What's new

### Stay signed in

Some users were getting logged out about once a day. This was caused by a timing issue when several
requests tried to refresh your session at the same moment. Jetstream now handles those simultaneous
requests safely, which keeps your session alive the way it should be.

### Select all when exporting objects

When you export object metadata, you can now select every object at once with a single toggle instead of
ticking each box. It's a small thing that saves a lot of clicks on orgs with many objects.

### Password resets show the rules up front

The password reset flow now displays the password requirements and validates your input against them as
you type — including a check that your password doesn't contain your own user information — so you get it
right the first time instead of submitting and bouncing back.

### Other fixes

- Saving object, field, and tab visibility permissions has clearer error messages and more reliable
  behavior.
- Styling fix for the profile / permission set popover button.
