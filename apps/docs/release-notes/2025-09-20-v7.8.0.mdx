## What's new

### Reliable large attachment uploads

Binary attachment loads now handle large files reliably. Much larger files upload without timeouts or running out of memory.

### Higher free-plan attachment limit

The attachment filesize limit for free-plan users was increased.

### Platform events publishing fix

Fixed an issue that could occur when publishing platform events.
